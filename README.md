# School-management-system-via-python
This is a little project for small scales schools for easy management 

# FEATURES
1. Studemt management system
2. teacher management system
3. uniform management system
4. Stationary management system
5. Fee Module
6. recipts management system
7. Teacher attendance management system
8. Attendance salary system
9. To-Do list
10. Removal of records prior to 1 year

# hoe to run / requirments
1. You must have python intall --> " https://www.python.org/downloads/ "
2. you should have tkinker install by running a command in cmd --> " **pip install tkinter** "
3. you are ready to run you software by running the **management_system.exe**
